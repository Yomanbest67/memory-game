# Card Memory Game
Memory card game built in React. 
### Rules
Click every card just once to win. 

## Live Page Deployment
Can be seen [here](https://yomanbest-memory-cards.netlify.app).

## Technologies used
- React
- JavaScript
- Webpack
- ES6 Modules
- npm
